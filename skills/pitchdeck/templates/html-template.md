# HTML Presentation Template (PDF-Optimized)

Reference architecture for pitch deck generation. Fixed 16:9 dimensions (1920×1080px per slide).

## Base HTML Structure

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Company Name — Pitch Deck</title>

    <!-- Fonts: Google Fonts via @import (embedded for PDF) -->
    <style>
        @import url('https://fonts.googleapis.com/css2?family=DISPLAY_FONT:wght@400;600;700;800&family=BODY_FONT:wght@400;500&display=swap');

        /* ===========================================
           CSS CUSTOM PROPERTIES (THEME)
           Change these to change the entire look
           =========================================== */
        :root {
            /* Colors — from chosen style preset */
            --color-bg: #FAFAF8;
            --color-surface: #F2F0EC;
            --color-text-primary: #1A1A1A;
            --color-text-secondary: #6B6B6B;
            --color-accent: #2D5BFF;
            --color-accent-muted: #D4DFFF;
            --color-border: #E5E3DF;

            /* Typography — FIXED px for PDF fidelity */
            --font-display: 'Display Font', sans-serif;
            --font-body: 'Body Font', sans-serif;
            --title-size: 72px;
            --h2-size: 56px;
            --h3-size: 42px;
            --body-size: 32px;
            --small-size: 24px;
            --hero-number-size: 120px;

            /* Spacing */
            --slide-padding: 80px;
            --content-gap: 40px;

            /* Animation (HTML preview only) */
            --ease-out-expo: cubic-bezier(0.16, 1, 0.3, 1);
            --duration-normal: 0.6s;
        }

        /* ===========================================
           BASE STYLES
           =========================================== */
        * { margin: 0; padding: 0; box-sizing: border-box; }

        /* --- PASTE slide-base.css CONTENTS HERE --- */

        /* ===========================================
           PRESET-SPECIFIC STYLES
           From chosen STYLE_PRESETS.md preset
           =========================================== */

        /* ... */

        /* ===========================================
           SLIDE TYPE STYLES
           From SLIDE_TYPES.md templates
           =========================================== */

        /* ... */
    </style>
</head>
<body>
    <!-- Optional: Progress bar (HTML preview only, hidden in print) -->
    <div class="progress-bar"></div>

    <!-- Optional: Navigation dots (HTML preview only) -->
    <nav class="nav-dots"><!-- Generated by JS --></nav>

    <!-- =========================================
         SLIDES
         Each <section class="slide"> = one page in PDF
         ========================================= -->

    <section class="slide cover-slide">
        <div class="slide-content">
            <img src="logo.png" alt="Company Logo" class="logo reveal">
            <h1 class="reveal">Company Name</h1>
            <p class="tagline reveal">One-line value proposition</p>
            <p class="author reveal">Founder Name — Month Year</p>
        </div>
    </section>

    <!-- More slides following VC_STRUCTURES.md order... -->

    <!-- =========================================
         JAVASCRIPT (HTML Preview Only)
         None of this affects PDF output
         ========================================= -->
    <script>
        /* ===========================================
           SLIDE PRESENTATION CONTROLLER
           Handles navigation + animations for HTML preview.
           PDF export ignores all JS — layout is pure CSS.
           =========================================== */
        class SlidePresentation {
            constructor() {
                this.slides = document.querySelectorAll('.slide');
                this.currentSlide = 0;
                this.scaleToFit();
                this.setupKeyboardNav();
                this.setupScrollNav();
                this.setupNavDots();
                this.setupProgressBar();
                this.showSlide(0);
                window.addEventListener('resize', () => this.scaleToFit());
            }

            /* Scale 1920×1080 slides to fit browser viewport */
            scaleToFit() {
                const scaleX = window.innerWidth / 1920;
                const scaleY = window.innerHeight / 1080;
                const scale = Math.min(scaleX, scaleY);
                document.body.style.transform = `scale(${scale})`;
                document.body.style.width = '1920px';
                document.body.style.height = `${1080 * this.slides.length}px`;
            }

            setupKeyboardNav() {
                document.addEventListener('keydown', (e) => {
                    if (e.key === 'ArrowDown' || e.key === 'ArrowRight' || e.key === ' ') {
                        e.preventDefault();
                        this.nextSlide();
                    }
                    if (e.key === 'ArrowUp' || e.key === 'ArrowLeft') {
                        e.preventDefault();
                        this.prevSlide();
                    }
                });
            }

            setupScrollNav() {
                let lastScroll = 0;
                let scrollTimeout;
                window.addEventListener('wheel', (e) => {
                    clearTimeout(scrollTimeout);
                    scrollTimeout = setTimeout(() => {
                        if (e.deltaY > 0) this.nextSlide();
                        else this.prevSlide();
                    }, 50);
                }, { passive: true });
            }

            setupProgressBar() {
                const bar = document.querySelector('.progress-bar');
                if (!bar) return;
                this.progressBar = bar;
                this.updateProgress();
            }

            setupNavDots() {
                const nav = document.querySelector('.nav-dots');
                if (!nav) return;
                this.slides.forEach((_, i) => {
                    const dot = document.createElement('button');
                    dot.className = 'nav-dot';
                    dot.addEventListener('click', () => this.showSlide(i));
                    nav.appendChild(dot);
                });
                this.navDots = nav.querySelectorAll('.nav-dot');
            }

            showSlide(index) {
                if (index < 0 || index >= this.slides.length) return;
                this.currentSlide = index;
                this.slides.forEach((s, i) => {
                    s.classList.toggle('visible', i === index);
                    s.style.display = i === index ? 'flex' : 'none';
                });
                if (this.navDots) {
                    this.navDots.forEach((d, i) =>
                        d.classList.toggle('active', i === index));
                }
                this.updateProgress();
                window.scrollTo(0, 0);
            }

            nextSlide() { this.showSlide(this.currentSlide + 1); }
            prevSlide() { this.showSlide(this.currentSlide - 1); }
            updateProgress() {
                if (this.progressBar) {
                    const pct = ((this.currentSlide + 1) / this.slides.length) * 100;
                    this.progressBar.style.width = pct + '%';
                }
            }
        }

        /* Show all slides for print (PDF export) */
        if (window.matchMedia('print').matches) {
            document.querySelectorAll('.slide').forEach(s => {
                s.style.display = 'flex';
                s.classList.add('visible');
            });
        } else {
            new SlidePresentation();
        }

        /* Before print: show all slides */
        window.addEventListener('beforeprint', () => {
            document.querySelectorAll('.slide').forEach(s => {
                s.style.display = 'flex';
                s.classList.add('visible');
            });
        });
    </script>
</body>
</html>
```

## Key Requirements

1. **Single self-contained HTML file** — all CSS/JS inline, no external dependencies except font @import
2. **Fixed 1920×1080px per slide** — `scaleToFit()` handles browser preview, @media print handles PDF
3. **Fonts via Google Fonts @import** — never system fonts. Include `display=swap`
4. **Every section: clear comment block** — `/* === SECTION NAME === */`
5. **Design must work WITHOUT JS** — CSS handles all layout. JS only for navigation/animation
6. **Print = all slides visible** — `beforeprint` event shows all, removes single-slide mode
7. **No external images** — embed as base64 or use file references in same directory

## Navigation Features (HTML Preview)

- Arrow keys (up/down, left/right)
- Space bar (next)
- Mouse wheel (scroll between slides)
- Click navigation dots
- Progress bar at top

## Image Handling

Use direct file paths for local viewing:
```html
<img src="assets/logo.png" alt="Logo" class="slide-image logo">
<img src="assets/screenshot.png" alt="Screenshot" class="slide-image">
```

```css
.slide-image {
    max-width: 100%;
    max-height: 600px;
    object-fit: contain;
    border-radius: 8px;
}
.slide-image.logo {
    max-height: 120px;
}
```

For PDF embedding (base64):
```html
<img src="data:image/png;base64,..." alt="Chart" class="slide-image">
```

## Code Quality

- **Comments:** Every section needs clear comments explaining customization
- **Accessibility:** Semantic HTML, keyboard navigation, ARIA labels
- **Print fidelity:** `print-color-adjust: exact` for background colors in PDF
